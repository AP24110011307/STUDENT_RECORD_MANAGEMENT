# Student Management System (C Project)

A simple role-based Student Management System written in C.  
The system supports **Admin**, **Staff**, and **Guest** roles with different permissions.

---

## Features

### **Admin**
- Add Student  
- Display Students  
- Search Student  
- Update Student  
- Delete Student  

### **Staff**
- Add Student  
- Display Students  
- Search Student  
- Update Student  

### **Guest**
- Display Students  
- Search Student
